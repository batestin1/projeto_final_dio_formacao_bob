# 🎯 Guia Rápido - Comandos Slash do Bob

Este projeto possui **3 comandos slash personalizados** que você pode usar diretamente no chat do Bob!

## 🚀 Comandos Disponíveis

### 1. `/trilha <tecnologia>`
Exibe o plano de estudos completo de uma trilha da DIO.

**Exemplos:**
```
/trilha Python
/trilha JavaScript
/trilha React
```

---

### 2. `/desafio <tecnologia> [nivel]`
Gera um desafio de código aleatório.

**Exemplos:**
```
/desafio Python
/desafio JavaScript Intermediário
/desafio Java Avançado
```

**Níveis:** Básico, Intermediário, Avançado

---

### 3. `/certificado "<nome>" "<tecnologia>"`
Gera um certificado de conclusão em Markdown.

**Exemplos:**
```
/certificado "João Silva" "Python"
/certificado "Maria Santos" "JavaScript"
```

⚠️ **Use aspas duplas!**
