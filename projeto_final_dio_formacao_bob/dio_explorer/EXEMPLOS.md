## 🎯 Comando `/desafio`

### Exemplo 1: `/desafio Python Básico`

```markdown
# 🎯 Desafio de Código - Python

## Calculadora de Fibonacci

**Dificuldade:** Básico | **Pontos:** 100 XP

---

### 📝 Descrição

Crie uma função que retorne o n-ésimo número da sequência de Fibonacci usando recursão ou iteração.

### ✅ Requisitos

1. Função que recebe um número inteiro n
2. Retorna o n-ésimo número de Fibonacci
3. Trate casos especiais (n <= 0)

### 💡 Dicas

- 💭 Fibonacci: F(n) = F(n-1) + F(n-2)
- 💭 F(0) = 0, F(1) = 1
- 💭 Considere usar memoization para otimizar

### 📊 Exemplos

**Exemplo 1:**
```
Entrada: fibonacci(5)
Saída: 5
```

**Exemplo 2:**
```
Entrada: fibonacci(10)
Saída: 55
```

### 🏆 Recompensas

- ⭐ 100 XP ao completar
- 🎖️ Badge de conclusão
- 📈 Progresso na trilha Formação Python Developer

### 🚀 Próximos Passos

1. Leia atentamente os requisitos
2. Planeje sua solução
3. Implemente o código
4. Teste com os exemplos fornecidos
5. Submeta sua solução na plataforma DIO

💪 **Boa sorte!** Use `/desafio Python Básico` para gerar outro desafio.
```

## 💡 Comando `/help`

```markdown
# 📚 DIO Explorer - Comandos Disponíveis

Sistema de comandos slash para interação com a plataforma DIO.

---

## /trilha

**Descrição:** Exibe o plano de estudos detalhado de uma trilha/tecnologia da DIO

**Uso:** `/trilha <tecnologia>`

**Exemplos:**
- `/trilha Python`
- `/trilha JavaScript`
- `/trilha React`
- `/trilha Java`

---

## /desafio

**Descrição:** Gera um desafio de código aleatório baseado na tecnologia e nível escolhidos

**Uso:** `/desafio <tecnologia> [nivel]`

**Exemplos:**
- `/desafio Python`
- `/desafio JavaScript Básico`
- `/desafio Java Intermediário`
- `/desafio React Avançado`

---

## /certificado

**Descrição:** Gera um certificado fictício de conclusão em formato Markdown

**Uso:** `/certificado "<nome>" "<tecnologia>"`

**Exemplos:**
- `/certificado "João Silva" "Python"`
- `/certificado "Maria Santos" "JavaScript"`
- `/certificado "Pedro Oliveira" "Java"`

---

## 💡 Dicas

- Use aspas duplas para argumentos com espaços
- Os comandos não são case-sensitive
- Você pode usar `/help` a qualquer momento

---

🚀 **Comece sua jornada de aprendizado agora!**
```

---
