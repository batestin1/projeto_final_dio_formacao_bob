# 🎬 Demonstração Prática dos Comandos Slash

Este arquivo demonstra como usar os comandos slash no chat do Bob.

---

## 🚀 Como Testar

Copie e cole cada comando abaixo diretamente no chat do Bob (um de cada vez):

---

## 📝 Demonstração 1: Explorar Trilha de Python

**Cole este comando no chat:**
```
/trilha Python
```

**O que você verá:**
- ✅ Informações completas da Formação Python Developer
- ✅ 12 módulos de conteúdo
- ✅ 3 badges disponíveis
- ✅ Plano de estudos em 3 fases
- ✅ 15.000 XP total

---

## 🎯 Demonstração 2: Gerar Desafio de JavaScript

**Cole este comando no chat:**
```
/desafio JavaScript Intermediário
```

**O que você verá:**
- ✅ Um desafio aleatório de JavaScript nível Intermediário
- ✅ Descrição detalhada do problema
- ✅ Requisitos técnicos
- ✅ Dicas de implementação
- ✅ Exemplos de teste
- ✅ 250 XP de recompensa

---

## 🎓 Demonstração 3: Gerar Certificado

**Cole este comando no chat (substitua "Seu Nome" pelo seu nome real):**
```
/certificado "Seu Nome Completo" "Python"
```

**O que você verá:**
- ✅ Confirmação de geração do certificado
- ✅ Informações do certificado (código, data, carga horária)
- ✅ Local onde o arquivo foi salvo
- ✅ Lista de conquistas desbloqueadas
- ✅ Total de XP conquistado

**Arquivo gerado em:**
```
projeto_final_dio_formacao_bob/dio_explorer/docs/certificados-emitidos/
```

## 💡 Dicas Importantes

### ✅ Faça Isso:
- Use aspas duplas no comando `/certificado`
- Especifique o nível no `/desafio` para resultados mais precisos
- Explore diferentes tecnologias com `/trilha`

### ❌ Evite Isso:
- Não esqueça a barra `/` no início
- Não use aspas simples no `/certificado`
- Não deixe argumentos vazios
