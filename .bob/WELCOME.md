# 🎉 Bem-vindo ao DIO Explorer!

Você acabou de abrir um projeto com **comandos slash personalizados** integrados ao Bob!

---

## 🚀 Comece Agora em 3 Passos

### 1️⃣ Abra o Chat do Bob
Clique no ícone do Bob ou pressione o atalho de teclado

### 2️⃣ Digite um Comando
Experimente este comando:
```
/trilha Python
```

### 3️⃣ Veja a Mágica Acontecer! ✨
O Bob retornará um plano de estudos completo da trilha de Python!

---

## 🎯 Comandos Disponíveis

### `/trilha <tecnologia>`
Exibe plano de estudos completo
```
/trilha Python
/trilha JavaScript
/trilha React
```

### `/desafio <tecnologia> [nivel]`
Gera desafio de código aleatório
```
/desafio Python Básico
/desafio JavaScript Intermediário
/desafio Java Avançado
```

### `/certificado "<nome>" "<tecnologia>"`
Gera certificado de conclusão
```
/certificado "Seu Nome" "Python"
```
