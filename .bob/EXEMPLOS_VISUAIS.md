## 🔍 Exemplo 4: Comando `/help`

### Entrada:
```
/help
```

### Saída:
```markdown
# 📚 DIO Explorer - Comandos Disponíveis

Sistema de comandos slash para interação com a plataforma DIO.

---

## /trilha

**Descrição:** Exibe o plano de estudos detalhado de uma trilha/tecnologia da DIO

**Uso:** `/trilha <tecnologia>`

**Exemplos:**
- `/trilha Python`
- `/trilha JavaScript`
- `/trilha React`
- `/trilha Java`

---

## /desafio

**Descrição:** Gera um desafio de código aleatório baseado na tecnologia e nível escolhidos

**Uso:** `/desafio <tecnologia> [nivel]`

**Exemplos:**
- `/desafio Python`
- `/desafio JavaScript Básico`
- `/desafio Java Intermediário`
- `/desafio React Avançado`

---

## /certificado

**Descrição:** Gera um certificado fictício de conclusão em formato Markdown

**Uso:** `/certificado "<nome>" "<tecnologia>"`

**Exemplos:**
- `/certificado "João Silva" "Python"`
- `/certificado "Maria Santos" "JavaScript"`
- `/certificado "Pedro Oliveira" "Java"`

---

## 💡 Dicas

- Use aspas duplas para argumentos com espaços
- Os comandos não são case-sensitive
- Você pode usar `/help` a qualquer momento

---

🚀 **Comece sua jornada de aprendizado agora!**
```
